# 注册表单设计描述

## 表单基本信息

页面名称：用户注册页  
设备：移动端（iOS / Android）  
当前版本：v2.3.1

## 字段列表（从上到下排列）

1. 姓名（文本输入框，placeholder：请输入真实姓名）
2. 手机号（数字输入框，placeholder：请输入手机号）
3. 密码（密码输入框，placeholder：请设置密码）
4. 确认密码（密码输入框，placeholder：请再次输入密码）
5. 图形验证码（输入框 + 验证码图片，placeholder：请输入验证码）
6. 注册按钮（主色调蓝色，宽度100%）

## 当前交互说明

- 所有字段均在用户点击「注册」按钮后统一校验，无实时反馈
- 校验不通过时，在页面顶部以红色 Toast 统一显示错误提示，文案为「部分信息填写有误，请检查后重试」
- 密码字段无强度提示，任意长度均可提交
- 确认密码不一致时，错误提示同上（不单独标红该字段）
- 图形验证码无刷新按钮，看不清只能退出重进
- 字段之间的间距：姓名与手机号之间 8px，手机号与密码之间 16px，密码与确认密码之间 8px，确认密码与验证码之间 24px，验证码与注册按钮之间 16px
- 键盘弹起时，注册按钮被遮挡，用户需手动滚动才能看到
- 表单无「已有账号？去登录」的引导跳转链接
