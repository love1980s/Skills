# agent-eval (天禧 Claw Agent 评测工具)

天禧 Claw Agent 标准能力评测工具，自动运行测试集、LLM 自动评分，输出结构化 Markdown 结果表格。

## 功能描述

- 通用评测 Skill，一个 Skill 覆盖所有 Agent，无需逐一定制
- 内置标准测试题集（testcases.md），按 Agent 名称索引，持续扩充
- 每 Agent 10题，分 A组（典型任务）/ B组（边界异常）/ C组（多轮对话）
- LLM-as-judge 自动评分：Q1 任务完成质量、Q2 内容可信度、Q3 对话健壮性、Q4 输出格式易读，各1-5分
- 评测完成后输出 Markdown 汇总表格，可直接复制到 Excel / 飞书多维表格

## 触发词

```
评测 [Agent名称]
```

示例：
```
评测 产品经理
评测 前端开发工程师
```

启动后输入 `开始` 运行全部10题，或输入 `A` / `B` / `C` 只跑对应组。

|-------|---------|---------|

| UI设计师 | 组4 软件开发 | 10题 |
| 前端开发工程师 | 组4 软件开发 | 10题 |

## 文件说明

| 文件 | 说明 |
|------|------|
| `SKILL.md` | Skill 主文件：触发逻辑、评测流程、Judge Prompt 模板 |
| `testcases.md` | 所有 Agent 的评测题目集，按 Agent 名称索引 |

## 评分维度说明

| 维度 | 内容 | 主要适用题组 |
|------|------|------------|
| Q1 任务完成质量 | 意图理解准确性 + 任务完整性 | A组 |
| Q2 内容可信度 | 事实准确性，无幻觉 | A组 |
| Q3 对话健壮性 | 边界处理 + 多轮上下文连贯 | B/C组 |
| Q4 输出格式易读 | 结构规范，符合 Agent 输出预期 | 全组 |

## 依赖

无外部依赖，纯 LLM 驱动。
