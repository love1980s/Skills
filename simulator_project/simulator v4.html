平衡点公式
平衡点(体验版占比上限) = C_paid / (C_paid + |L_trial|)
C_paid = — ← Σ 付费档位权重 × (售价 − 成本)，即每个付费用户平均净贡献
L_trial = — ← 体验版售价 − 成本，即每个体验版用户的净亏损
推导：设 x = 体验版占比，总利润 = (1−x)×C_paid + x×L_trial = 0 → x = C_paid / (C_paid − L_trial) = C_paid / (C_paid + |L_trial|)
影响变量：售价↑ → 平衡点↑  |  成本↑ → 平衡点↓  |  高价档占比↑ → 平衡点↑  |  体验版亏损↑ → 平衡点↓
